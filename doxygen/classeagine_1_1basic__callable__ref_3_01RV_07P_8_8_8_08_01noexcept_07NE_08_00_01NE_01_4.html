Detailed Description

template<typename RV, typename... P, bool NE>
class eagine::basic_callable_ref< RV(P...) noexcept(NE), NE >

Implementation of class template storing a reference to a callable object.

See also
callable_ref

Unlike std::function basic_callable_ref does not do any conversion on the parameters or return values, the referenced callable must match, and it does not copy the callable, it must remain in scope while the reference is used.
